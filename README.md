# noctalia-plugins
Collection of my own plugins for Noctalia-shell

- **Color hyprpicker**: A color picker plugin that extends Noctalia's base one with hyprpicker. Paste several color codes to your clipboard,
  and browse recent colors list.
- **Hyprshot menu**: A GUI to do several kinds of screenshots with hyprshot.
- **Radio player**: A radio player widget that lets you listen to any radio station online from a custom list.
